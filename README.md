# FFMPEG Simple GUI

Remember, need the ffmpeg installation in OS Path.

## Without .EXE

### Install Dependency
```bash
python -m pip install -r requirements.txt
```
### Execute
```bash
python main.py
```

## With .EXE (Use Pyinstaller)
### Create with Pyinstaller
```bash
pyinstaller --onefile main.py --name ffmpeggui
```
